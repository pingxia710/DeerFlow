# Durable project guidance

Keep durable project guidance limited to decisions that remain useful beyond the current task: a confirmed goal or boundary, a known debt, a safety constraint, or a lesson that prevents a repeat failure.

The Chair decides whether a note is worth retaining after reading the relevant
planning angles, execution notes, and review findings. Runtime stores references
and facts but does not promote suggestions into policy.

Temporary task notes do not need a prescribed account, template, or approval path.
